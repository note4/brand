## Local update favicon

### Python dependency initialization

Run this command the first time, and then as needed. You should also run it if the `requirements.txt` file is updated.

```bash
py -m pip install --upgrade pip
py -m pip install -r requirements.txt
```

```bash
py update_favicons.py
```
